# GP-11 C++ Homework Collection

This repository contains homework assignments for the C++ console programming class.

## Projects Overview

- **GP11-01?Commemorative Post (記念カキコミ)**  
  - A console program that displays a student’s name, age, and hobbies in Japanese.

- **GP11-02?Simple Calculator (簡易電卓)**  
  - Reads two integers and outputs their sum, difference, product, quotient, and remainder.  
  - Handles division by zero gracefully.

- **GP11-03?Number Guessing Game (数当てゲーム)**  
  - User tries to guess a pseudo-randomly generated number.  
  - Feedback until correct guess.  
  - Two modes:  
    - “Higher/Lower” hints.  
    - “Digit-by-digit” correctness hints.

- **GP11-04?Odd Magic Square (奇数魔法陣)**  
  - Generates an odd-sized magic square (3×3, 5×5, etc.).  
  - Displays the filling process step by step on each key press.

- **GP11-05?Name Battler (ネームバトラー)**  
  - Creates characters deterministically from input names.  
  - Supports multiple characters and battle simulations:  
    - AI vs AI, AI vs Player, Player vs Player.  
    - Battle Tower mode.  
  - Character growth system: characters earn experience and level up.

# GP-11 C++ 課題コレクション

このリポジトリには、C++ コンソールプログラミングの授業課題を収録しています。

## プロジェクト概要

- **GP11-01?記念カキコミ (Commemorative Post)**  
  - コンソールプログラム  
  - 学生の名前・年齢・趣味を日本語で自己紹介表示

- **GP11-02?簡易電卓 (Simple Calculator)**  
  - 2つの整数を読み取り  
  - 和・差・積・商・余りを出力  
  - ゼロ除算に対応

- **GP11-03?数当てゲーム (Number Guessing Game)**  
  - 乱数で生成された数をユーザーが予想  
  - 答えが出るまでヒントを表示  
  - モードは二つ：  
    - 答えより「大きい／小さい」ヒント  
    - 各桁ごとの正誤ヒント

- **GP11-04?奇数魔法陣 (Odd Magic Square)**  
  - 奇数サイズ（3×3、5×5 など）の魔法陣を生成  
  - キー入力ごとに埋める過程をステップ表示

- **GP11-05?ネームバトラー (Name Battler)**  
  - 入力された名前からキャラクターを生成（決定論的）  
  - 複数キャラクターの作成・対戦シミュレーション：  
    - AI対AI、AI対プレイヤー、プレイヤー対プレイヤー  
    - バトルタワー  
  - 育成システム：対戦で経験値を獲得しレベルアップ
